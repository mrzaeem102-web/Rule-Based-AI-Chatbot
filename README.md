# 🤖 Rule-Based AI Chatbot

A simple and interactive Rule-Based AI Chatbot built using HTML, CSS, and JavaScript. This chatbot responds to predefined commands using if-else decision-making logic and demonstrates the fundamentals of Artificial Intelligence.

---

## 📌 Project Overview

This project simulates a basic AI chatbot that can interact with users through predefined responses. It is designed to showcase beginner-level AI concepts, JavaScript logic building, DOM manipulation, and responsive web design.

---

## ✨ Features

- Interactive chat interface
- Rule-based AI responses
- Sidebar navigation
- Chat statistics panel
- Real-time message counter
- Enter key support
- Clear chat functionality
- Responsive user interface
- Modern and clean design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📋 Supported Commands

| Command | Description |
|----------|------------|
| hello | Greeting |
| hi | Greeting |
| hey | Greeting |
| about | About chatbot |
| project | Project information |
| features | List chatbot features |
| ai | Information about AI |
| chatbot | What is a chatbot |
| internship | Internship-related info |
| what can you do | Chatbot capabilities |
| help | Display commands |
| thank you | Appreciation response |
| bye | Exit conversation |
| exit | Exit conversation |
